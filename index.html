CSS Grid Layout - Display Grid

Aula 01: display grid

Ao usar o display grid, as configurações de container serão atribuídas para a contrução de um layout dinâmico

  • Display Grid: Padrão
  • Inline-Grid: O tamanho item será conforme o maior item

Para começar a definir a grade, usa-se: Grid Template Columns (usa-se Nrf ou tamanho de cada grade) e Grid Template Row (Define-se o tamanho de cada linha, quando não, será auto)

Aula 02: Grid Gap

Espaços entre colunas e linhas

Parâmetros:

  • Columns-gap: define o espaço entre as colunas
  • Row-gap: define o espaço entre as linhas
  • Gap: resume o row-gap e columns-gap no mesmo parâmetro (valor único vai valer para os os dois)

Aula 03: Posição inicial e final de elementos dentro da grade

Parâmetros:

  • grid-columns-start: o item inicia na coluna n
  • grid-columns-end: o item termina na coluna n (vai até o valor, mas não o ocupa)
  • grid-columns: versão reduzida de grid-columns-start e grid-columns-end

    • Pode-se usar no columns: 1/ span 3 (o item será espandido em 3 colunas)

  • grid-row-start: o item inicia na linha n
  • grid-row-end: o item termina na linha n
  • grid-row: versão reduzida de grid-row-start e grid-columns-end
